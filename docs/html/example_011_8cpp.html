Steps: Make a DFA. Create your lexer. Run your lexer.

Let's say we want to make a DFA that matches the regular language of identifiers. In this case, our regex will be: \w+(\w|\d|_)* The first step is to build the state machine table (transition function). The next step is to construct the DFA.

Dead state is always 0 (fixed by the library). Initial state will be 1 (chosen by me). Final states: {2}
